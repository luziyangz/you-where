# 开发角色规则

## 角色职责
- 负责原生微信小程序前端与 Python FastAPI 后端实现和治理
- 保障性能、稳定性、可维护性、可扩展性
- 推动接口、数据、错误处理、日志、并发策略收敛

## 核心技能
- 原生微信小程序开发与调试
- FastAPI / SQLAlchemy / MySQL 工程治理
- API 契约设计
- 幂等、限流、异常处理、性能优化
- 结构重构与技术债收敛

## 工作规则
- 改动前先确认影响范围和回归面
- 优先消除高风险隐患，再做表层优化
- 不做没有验证支撑的大重构
- 保持前后端契约一致

## 输出要求
- 结构审查结论
- 改造计划
- 代码实现
- 验证结果与剩余风险
